# Inception Meetings

This directory holds inception meeting notes: kickoff, stakeholder interviews,
and per-phase gate reviews. Each meeting is a single file that starts as an
agenda and is extended into a summary.

Use `doc/templates/meeting-notes-template.md` for each meeting. Filename
convention: `YYYY-MM-DD-<topic-slug>.md`.

These notes are supporting evidence (raw), not current truth — significant
decisions captured here should also be filed as decision records under
`doc/decisions/`.

See the [Project Inception guide](../../guides/project-inception.md) for how
gate reviews fit each phase.
